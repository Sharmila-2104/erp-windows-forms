# ERP Application

ERP system developed using C# Windows Forms.

## Modules
- Inventory (Item Master)
- Purchase (Vendor Master)
- General Ledger
- Sub Masters

## Technologies
- C# (.NET Framework)
- Windows Forms
- ADO.NET
- SQL Server
- DAO / DTO Architecture
  
## Architecture
- DAO (Data Access Object)
- DTO (Data Transfer Object)
- Layered architecture

## Features
- CRUD operations
- DataGridView-based forms
- Validation and alerts
- Modular ERP design
